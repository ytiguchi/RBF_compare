# 資金調達オプション詳細比較・完全収益シミュレーション指示書

本資料は、提供額1,000万円を基準とし、全てのオプションを**「同じ期間（12ヶ月）」「端折りのない月次データ」「元本・金利の完全分離」**という極めて厳格な基準で再整理したものです。

---

## 🛠 HTML作成・図表描画ルール (Mandatory)
情報を省略せず、比較可能性を最大化するために以下のルールを厳守してください。

1.  **情報の非省略**: 
    *   月次のタイムチャートにおいて「...」による省略を一切禁止する。全12ヶ月（月0〜月12）の全行を必ず記述すること。
2.  **パラメーターの一貫性と明示**:
    *   全てのセクションの冒頭に「初期投資額」「期間」「利率/手数料率」を構造化して明記すること。
3.  **回収内訳の完全分離**:
    *   全てのテーブルに「元本回収分」と「収益（金利/手数料）分」の列を設け、それぞれの合計が提供額および総収益と一致することを検証可能にすること。
4.  **全セクションへの個別グラフ配置**:
    *   各商品のセクション内に、その商品の特徴（残高推移、累積収益、回収内訳）を示す個別グラフを必ず配置すること。サマリーグラフだけで済ませないこと。

---

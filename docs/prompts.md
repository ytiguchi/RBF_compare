## 補足（内容の根拠）

### なぜ「適法に設計可能」と言えるか

1. **既存の類似商品がある**
   - 銀行・信金の「据置期間付きローン」「利息のみ期間」
   - バルーンローン（満期一括返済型）
   - 当座貸越・リボルビング

2. **上場会社の強み**
   - 貸金業登録を取得する体制・資金力がある
   - コンプライアンス体制が整っている
   - 法務・顧問弁護士のチェックが入る

3. **設計パターン**
   - 案A: 利息のみ＋満期で元本一括（バルーン型）→ 満期ごとに更新審査
   - 案B: 利息のみ＋任意償還権 → 借り手がいつでも返済可能

### 必須条件の理由

| 条件 | 理由 |
|------|------|
| 貸金業登録 | 元本を渡して利息を受け取る = 金銭消費貸借 = 貸金業法の対象 |
| 期限を設ける | 「永久」「無期限」は規制リスク・消費者保護リスクが高い |
| 上限金利遵守 | 利息制限法: 10万未満20%、100万未満18%、100万以上15% |
| 出口の設計 | 満期時の更新審査 or 任意償還で「終わり方」を明確化 |
